My project is an AI-powered Policy Conflict and Staleness Detector. Large organisations have many policies written by different departments, and some policies may contradict each other or become outdated. My system uploads and analyses policy documents using the Groq API. It extracts important obligations, compares similar rules, detects conflicts and redundancies, identifies policies older than 18 months, and flags deprecated technologies. The findings are displayed in a Flask-based dashboard with graphs and reports.
