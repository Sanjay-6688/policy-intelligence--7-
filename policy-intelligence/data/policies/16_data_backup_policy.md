# Data Backup Policy

## 1. Purpose
Ensures recoverability of corporate data in the event of loss or corruption.

## 2. Scope
Applies to all production systems storing business-critical data.

## 3. Requirements

### 3.1 On-Prem Backup Requirement
All production systems must maintain a daily on-premises backup in addition to any other backup mechanism, retained for a minimum of 30 days.

### 3.2 Backup Testing
Backup restoration must be tested quarterly.

### 3.3 Encryption
Backup media must be encrypted at rest.
